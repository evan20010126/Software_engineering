Customer's web introduction


Login Page

進入使用者 website 後即可看到登入畫面，畫面採簡潔的設計，若無帳號可進行註冊，而無登入的訪客也可以從上方的導覽列進入觀看菜單，但功能有所限制，無法訂餐僅可閱讀。

Browse the menu & Order

在即時點餐的頁面中，可以瀏覽商家目前供應的菜色，若為使用者則可進一步加入商品至購物車，並確認餐點後結帳。

View order

在這個頁面中登入的使用者可以看到，目前結帳過後商家還沒有確認過的訂單，與正在製作的餐點訂單，與曾經購買過的歷史訂單。

Account Setting

此頁面可以進行帳號的相關設定，與管理，能自由的變更帳號內的一些相關訊息。

Preference Setting

在這個頁面中，登入的使用者可以設定使用者各自的偏好，使訂單會註記此資訊給商家，客製化的提供餐點。
